# Science

The completion point is sustained lab consumption, not packs in inventory or a
chest.

- Work backward from the lab through science packs, intermediates, plates, ore,
  fuel, and power.
- For each ingredient, find the first missing or manual dependency and automate
  that edge before expanding another branch.
- Ensure assembler outputs flow onward instead of accumulating at disconnected
  endpoints.
- Start research and prove progress across the observation window with no
  character delivery.
- Keep one verified science tier running before selecting the next tier.
