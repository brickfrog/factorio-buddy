# Bootstrap

Bootstrap exists only to create the machinery needed for autonomous flow.

- Choose the smallest manual batch that unlocks the next machine-mediated edge.
- State which dependency the batch will eliminate before performing it.
- Build that edge immediately after the batch; do not replenish the same
  consumer manually a second time.
- End bootstrap when extraction, fuel, processing, and output movement can
  continue while the character stands still.
- Start a clean verification window after the final manual action.
